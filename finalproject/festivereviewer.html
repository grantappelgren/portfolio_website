FESTIVE REVIEWER

About the Project

Festive Reviewer is a blog website where I share my experiences attending different music festivals. For a college project, I was tasked with creating a website of my choice focused around something I am passionate about, such as music festivals. The intended audience is any festival-goer interested in learning information, pros, cons, etc. about these events. Pre-production was required to include a color scheme, typography and wireframes, while production needed to consist of logo design and responsive page design that ultimately resulted in a functional website containing multimedia, text, and interactive elements.

Visual Identity

Music Festivals are loud, bright, and often times chaotic. That said, I wanted to design a very straightforward interface that did not interfere with the subject-matter. Blogs should have the ability to be added upon, so a format that would be able to easily adjust as new content is added was important. Simple navigation and a consistent way of displaying my reviews of each festival across pages would help users compare festivals to one another.

Visual Design

A design ranging between 4, 2, and 1-column depending on the user's screen size creates an organized look-and-feel, allowing for the content to be the main attraction. Similarly, colors were selected to be calm and easy on the eyes as users read text. The landing page lists festivals as buttons, providing a way for users to interact with the site and access individual festival pages. On each of these pages, the same four columns (Production, Variety, Atmosphere, Return?) are featured and assigned letter or Yes/No grades, developing a way to rank festivals by section. Tints of green and orange are used to semantically represent past and future festivals as well as their grades. Overall, the website is clean, inviting, and a nice place for festival-goers to visit.
