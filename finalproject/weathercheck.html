WEATHER CHECK

About the Project

Weather Check is a single-page web-application that can be used via desktop or mobile device to check the weather. The application is powered by the Open Weather Map API. For a college project, I was tasked with creating the Layout and Design for the application. Pre-production was required to include a mood board and color scheme, with production needing to include logo and logotype, mobile app icon and page design. The objective was to design an application with an inviting and straightforward interface that would appeal to the public and encourage use. Inspired by YouTube instructor Dave Gray, I decided to further design and develop the application into one that is fully functional and operational. With this being a web rather than mobile-application, I chose to only display certain elements such as the logo, mobile app icon, and splash page here instead of within the current product. Each of the aforementioned elements would be included in a native or hybrid mobile version of the application.

Visual Identity

To facilitate ease of use, creating a simplistic design was key. Weather has a look and feel, so I wanted to incorporate both of those elements through relevant colors and icons. Designing areas that are sleek and encourage the user to interact and have an enjoyable experience was also a point of emphasis.

User Experience

Visual Design

Weather is very complex, so to prevent a cluttered screen, I made sure to selectively include only frequently-used information in the application. The minimalistic design uses basic shapes and recognizable weather identifiers with the purpose of creating familiarity and a comfortable experience. Warm and cool colors and gradients of those colors were chosen to effectively generate a sense of the look and feel of given weather types. Smooth current-view and home-view buttons, along with save-as-home, refresh, and imperial-to-metric degree toggle capabilities allow users to conveniently navigate and customize the tool with ease. Overall, the application meets the goal of being both informational and pleasant to use.
